Pantalla Principal

Desde esta ventana puedes acceder a todas las funcionalidades de la aplicación Ibiza Event Manager.

Funciones principales

  • Gestionar Eventos, Artistas y Reservas.
  • Cambiar el fondo de la interfaz para personalizar la apariencia.
  • Acceder al Calendario de Eventos para ver fechas importantes.
  • Utilizar la barra de búsqueda para encontrar eventos o artistas rápidamente.
  • Acceder a la Configuración para ajustes personalizados.
  • Cerrar sesión y volver a la pantalla de inicio de sesión.

Atajos de teclado

  • F1 - Abrir la ayuda.
  • Ctrl + E - Abrir la sección de eventos.
  • Ctrl + A - Abrir la sección de artistas.
  • Ctrl + R - Abrir la sección de reservas.
  • Ctrl + L - Cerrar sesión.
