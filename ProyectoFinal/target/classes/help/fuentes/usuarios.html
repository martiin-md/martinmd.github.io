Ayuda - Gestión de Usuarios

Esta ventana permite gestionar los usuarios del sistema. Desde aquí, puedes agregar, modificar y eliminar usuarios registrados.

Funciones principales:

  • Añadir Usuario: Permite registrar un nuevo usuario. Debes ingresar su nombre, correo electrónico, contraseña y tipo de usuario.
  • Modificar Usuario: Selecciona un usuario de la lista y edita sus datos. Puedes cambiar el nombre, correo, contraseña y tipo de usuario.
  • Eliminar Usuario: Elimina el usuario seleccionado. Se solicita una confirmación antes de proceder.

Instrucciones de Uso:

Para interactuar con la gestión de usuarios:

  1. Selecciona un usuario en la tabla si deseas modificarlo o eliminarlo.
  2. Presiona el botón correspondiente según la acción que deseas realizar.
  3. Si añades o modificas un usuario, completa los datos requeridos en la ventana emergente.
  4. Confirma la acción cuando se te solicite.

Nota:

Asegúrate de introducir información válida. Los cambios realizados se guardarán en la base de datos.
