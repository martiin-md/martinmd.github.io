Bienvenido a la ayuda de Ibiza Event Manager

Esta aplicación permite gestionar eventos en Ibiza, administrando artistas, reservas y otros aspectos relacionados.

La aplicación consta de varias ventanas, cada una con su propia página de ayuda:

  • Presentación
  • Inicio de Sesión
  • Pantalla Principal
  • Gestión de Eventos
  • Gestión de Artistas
  • Gestión de Reservas
